# 7.0.0 — Dashboard

## 7.1.0 Overview

The dashboard provides an at-a-glance overview of the user's Anthropic resources — agent count, environment count, active sessions, and recent activity.

## 7.2.0 Page

| Route | Purpose |
|-------|---------|
| `/dashboard` | Overview with aggregate metrics and recent sessions |

## 7.3.0 Metrics Displayed

- Total agents (active)
- Total environments (active)
- Total sessions
- Active sessions (status: running or idle)
- Recent sessions with status badges

## 7.4.0 Data Source

All metrics are fetched from the Anthropic API at page load via server-side data loading. No local caching — always reflects current state.

## 7.5.0 Design Decision

The dashboard is read-only and aggregation-focused. It does not duplicate CRUD functionality available in domain-specific pages — it links to them.
