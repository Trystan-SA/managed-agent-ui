# 7.0.0 — Dashboard

## 7.1.0 Overview

The dashboard provides an at-a-glance overview of Anthropic resources — agent count, environment count, active sessions, and recent activity. Since the Anthropic API key is global (shared by all users), the dashboard shows the same resource data regardless of which user is viewing it.

## 7.2.0 Page

| Route | Purpose |
|-------|---------|
| `/dashboard` | Overview with aggregate metrics and recent sessions |

## 7.3.0 Metrics Displayed

- Total agents (active)
- Total environments (active)
- Total sessions
- Active sessions (status: running or idle)
- Recent sessions with status badges

## 7.4.0 Data Source

All metrics are fetched from the Anthropic API at page load via server-side data loading using the global API key. No local caching — always reflects current state. If no API key is configured, the dashboard displays an error state prompting an admin to add one.

## 7.5.0 Design Decision

The dashboard is read-only and aggregation-focused. It does not duplicate CRUD functionality available in domain-specific pages — it links to them.
