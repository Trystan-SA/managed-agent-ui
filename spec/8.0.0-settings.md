# 8.0.0 — Settings & Preferences

## 8.1.0 Overview

User settings management — API key rotation, theme selection, and default resource preferences.

## 8.2.0 Data Model

### 8.2.1 API Keys

| Column | Type | Notes |
|--------|------|-------|
| id | uuid | PK |
| user_id | uuid | FK → users, unique |
| encrypted_key | bytea | AES-256-GCM encrypted |
| iv | bytea | Initialization vector |
| label | text | Optional display label |
| created_at | timestamp | |
| updated_at | timestamp | |

### 8.2.2 User Preferences

| Column | Type | Notes |
|--------|------|-------|
| id | uuid | PK |
| user_id | uuid | FK → users, unique |
| theme | text | `'light'` or `'dark'` |
| default_agent_id | text | Optional default agent |
| default_env_id | text | Optional default environment |
| created_at | timestamp | |
| updated_at | timestamp | |

## 8.3.0 API Routes

| Method | Route | Description |
|--------|-------|-------------|
| PUT | `/api/settings/api-key` | Update/rotate API key |
| DELETE | `/api/settings/api-key` | Remove API key |

## 8.4.0 Page

| Route | Purpose |
|-------|---------|
| `/settings` | API key management, theme toggle, account info |

## 8.5.0 Features

- **API key rotation**: Replace encrypted key without re-registering
- **Theme toggle**: Dark/light mode persisted in `user_preferences`
- **Default agent**: Pre-select agent when creating new sessions
- **Default environment**: Pre-select environment when creating new sessions

## 8.6.0 Theme System

- Theme preference stored in DB (`user_preferences.theme`)
- Applied via `[data-theme]` attribute on `<html>` element
- Svelte store (`theme.ts`) syncs client state
- Default: dark theme
